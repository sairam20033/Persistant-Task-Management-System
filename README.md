# Persistent Task Management System

A Python-based command-line application that helps users manage daily tasks efficiently. The system allows users to create, update, delete, and track tasks while storing data permanently using JSON files.

## Features

- Add new tasks
- Update existing tasks
- Delete tasks
- View all tasks
- Persistent task storage using JSON
- Simple command-line interface

## Technologies Used

- Python
- JSON
- File Handling

## How It Works

The application stores task information in a JSON file, allowing users to access their tasks even after restarting the program. Users can manage tasks through menu-driven options and keep track of their progress.
